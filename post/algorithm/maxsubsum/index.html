求最大子段和

双指针思想，先前缀和，再在前缀和上跑双指针，前面的指针记录最小值，每次更新pre在 $[1,i]$ 的最小值，用 $pre_i$ 减去之，更新到 $ans$ 即可。复杂度：$O(n)$

vector<ll> pre(n+1);
FOR(i,1,n){cin>>pre[i];pre[i]+=pre[i-1];}
ll mx=0,ans=0;
FOR(i,1,n){//记得看看题目允不允许子段为空
    mx=min(mx,pre[i-1]);
    ans=max(ans,pre[i]-mx);
}
//然后ans即为所求
